ΔΗΜΟΣΙΑ PROJECT — Αναθεωρημένο πακέτο HTML/CSS

Δομή:
- assets/css/dimosia.css: κοινό εξωτερικό CSS για όλες τις σελίδες.
- index.html: απλή αρχική σελίδα πλοήγησης.
- elaxisto-kostos-diaviosis.html: νέα αναθεωρημένη σελίδα για τη Μελέτη Ελάχιστου Κόστους Διαβίωσης.
- mouseio14.html
- meleti-zimias.html
- fakelos-koinonikis-prostasias.html
- atomikos-fakelos-trapezikis-zimias.html

Κεντρική γραμμή διατύπωσης:
- Δεν υποσχόμαστε αποτέλεσμα, αποζημίωση, διαγραφή χρέους ή δικαστική επιτυχία.
- Παραδίδουμε τεκμηριωμένο φάκελο προς αξιοποίηση από πολίτη, δικηγόρο, λογιστή ή άλλο αρμόδιο επαγγελματία.
- Δεν αποκαλύπτονται εργαλεία, λογισμικό, πρότυπα, βάσεις, αυτοματισμοί, πηγαίος κώδικας ή εσωτερική μεθοδολογία.
- Η διατύπωση για τη θεσμική βαρύτητα είναι: ο φάκελος δεν εγγυάται αποδοχή, αλλά όταν τίθεται νόμιμα ενώπιον αρμόδιας αρχής ή δικαστηρίου πρέπει να αξιολογείται με ειδική και αιτιολογημένη κρίση.

Σημείωση:
Τα τρία αρχεία για τον Φάκελο Θετικής Κοινωνικής Προστασίας ήταν όμοια/παράλληλα. Στο πακέτο κρατήθηκε ένα τελικό καθαρό αρχείο.


Πλοήγηση:
Όλες οι σελίδες έχουν κοινή επάνω μπάρα πλοήγησης. Σε κινητό, τα κουμπιά της μπάρας σύρονται οριζόντια αριστερά/δεξιά. Η αρχική σελίδα είναι το index.html.
