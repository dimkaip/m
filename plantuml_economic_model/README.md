# PlantUML - Οικονομικό Μοντέλο Συνεργασίας

Το πακέτο περιέχει δύο PlantUML διαγράμματα:

1. `01_oikonomiko_montelo_synergasias.puml`  
   Αναλυτικό διάγραμμα ροής για το οικονομικό μοντέλο συνεργασίας πολιτών, δικηγόρων και τεχνικής/οικονομικής ομάδας.

2. `02_diagramma_rois_axias.puml`  
   Απλούστερο διάγραμμα ροής αξίας, κατάλληλο για παρουσίαση.

## Εκτέλεση

Με εγκατεστημένο PlantUML:

```bash
plantuml 01_oikonomiko_montelo_synergasias.puml
plantuml 02_diagramma_rois_axias.puml
```

Για SVG:

```bash
plantuml -tsvg 01_oikonomiko_montelo_synergasias.puml
plantuml -tsvg 02_diagramma_rois_axias.puml
```
