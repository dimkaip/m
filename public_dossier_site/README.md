# Δημόσιο HTML ντοσιέ

Αυτός ο φάκελος περιέχει μόνο το δημόσιο υλικό που μπορεί να ανέβει στο δίκτυο.

## Αρχεία

- `index.html`: η δημόσια σελίδα παρουσίασης.
- `assets/style.css`: βασική μορφοποίηση.

## Ανέβασμα στο site

Ανεβάστε το `index.html` και τον φάκελο `assets/` στον ίδιο κατάλογο του server.

Δεν περιλαμβάνονται συμβάσεις, έντυπα υπογραφών, πρωτόκολλα παραλαβής/παράδοσης ή εσωτερικά φύλλα ελέγχου ποιότητας. Αυτά πρέπει να δίνονται μόνο κατά την έναρξη συνεργασίας.
